[![Anaconda Platforms](https://anaconda.org/hep-forge/minio-cpp/badges/platforms.svg)](https://anaconda.org/hep-forge/minio-cpp)

Feedstock for [minio-cpp](https://github.com/minio/minio-cpp) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

MinIO C++ Client SDK for Amazon S3 Compatible Cloud Storage

## Install

```bash
conda install -c hep-forge -c conda-forge minio-cpp
```
